# Homepage Redesign Spec

## New section order
1. Hero: Astra-led cinematic intro
2. Ecosystem identity: JoinHavn / Astra / HavnAI relationship
3. Astra spotlight: world, play, rewards, immersion
4. HavnAI engine: create, own, trade, connect
5. Core loop: Create → Own → Earn → Play
6. Surface grid: Generator, Library, Marketplace, Credits, Run a Node, Astra
7. Network/operator layer
8. Final CTA

## Design rules
- emotional first, explanatory second
- fewer dashboard surfaces above the fold
- Astra leads the feeling, HavnAI explains the system
- no install command in hero
- no dense stats as first impression
- dark premium sci-fi styling with restrained glow and motion

## Hero copy direction
Headline options should frame Astra as the emotional front door and HavnAI as the system underneath.

## CTA structure
Primary: Enter Astra
Secondary: Explore HavnAI
Tertiary: Run a Node

## Page objective
A visitor should understand within the first few sections:
- JoinHavn is the umbrella
- Astra is the flagship world
- HavnAI is the engine
- the ecosystem loop is Create → Own → Earn → Play
