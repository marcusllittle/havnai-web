# HavnAI Ecosystem Map

_Last updated: 2026-04-09_

---

## What HavnAI Is (One Sentence)

> HavnAI is a decentralized AI creation network where you generate content, own your outputs, earn for contributing compute, and spend or trade inside a live economy.

---

## The Four Pillars

| Pillar | What It Is | Who It's For |
|---|---|---|
| **Network** | Decentralized GPU grid that runs AI jobs | Operators, developers |
| **Creator App** | Generator, Library, Marketplace | Creators, collectors, traders |
| **Economy** | Credits (spend), HAI (earn/own), wallet (identity) | Everyone |
| **Astra Valk** | Flagship game world where assets and rewards have meaning | Players, collectors |

---

## Roles

### Creator
- Connects wallet
- Buys credits
- Generates images/video via Generator
- Saves to Library
- Lists on Marketplace
- Earns HAI rewards when outputs are popular or traded

### Operator
- Has a GPU machine
- Runs the node installer
- Serves generation jobs
- Earns tracked HAI rewards per job
- Builds operator history and reputation

### Collector / Trader
- Browses Marketplace
- Buys assets using credits
- Owns generation output on-chain
- Can relist, gift, or use in Astra Valk

### Player
- Plays Astra Valk
- Earns in-game credits from runs
- Unlocks pilots, outfits, gear tied to HavnAI asset economy
- Wallet connection bridges game rewards to HavnAI network

---

## The Two Currencies (Keep These Clear)

### Credits
- **What**: App spending unit
- **How to get**: Buy on pricing page, earn from game runs, bonus from promotions
- **Spend on**: Generator jobs, Marketplace purchases
- **Not crypto**: Managed off-chain, no wallet required to use

### HAI Token
- **What**: Network-layer economic token
- **How to get**: Operator rewards for serving jobs, creator royalties, network participation
- **Spend on**: Protocol staking, future governance, premium features
- **Is crypto**: Requires wallet, tracked on-chain

> **Simple rule:** Credits = spend. HAI = earn/own.

## User Flows

### New Creator (no wallet)
```
Home → Generator (no wallet) → Generate image → Save to Library → done
```

### Creator with wallet
```
Home → Connect wallet → Buy credits → Generator → Save to Library → List on Marketplace → Earn on trades
```

### Operator
```
Home → Join / Run a Node → Install → Serve jobs → Earn tracked HAI → See on Dashboard/Analytics
```

### Collector
```
Marketplace → Browse → Buy with credits → Saved to collection → Relist or use in Astra
```

### Player
```
Astra Valk → Play → Earn credits → Spend on outfits/gear → Connect wallet → Bridge rewards to HavnAI
```

### Full loop (power user)
```
Generate → Save → List → Sell → Earn HAI → Spend or stake → Run node → Earn more → Play Astra
```

## Where the Flow Breaks Today

### 1. Home tries to serve everyone at once
Right now: hero mixes "generate content" and "run a node" in the same block.  
Should be: clear entry points by role early on the page.

### 2. "Join" reads like universal onboarding
Right now: Join is the node installer guide.  
Should be: relabeled "Run a Node" or "Become an Operator" in the nav.

### 3. Credits vs HAI is not explained clearly
Right now: some pages mention both without explaining the distinction.  
Should be: one plain explanation visible before any economic action.

### 4. Generator success has no obvious next step
Right now: generate → done.  
Should be: generate → save → share/list/use in Astra.

### 5. Library feels like a local job list
Right now: shows your recent jobs, status, download.  
Should be: your creative inventory and the hub for listing, sharing, using in game.

### 6. Astra Valk's role in the ecosystem is implicit
Right now: it's a separate game that earns credits.  
Should be: explicitly the consumer world where HavnAI outputs and rewards gain context and meaning.

### 7. Marketplace lacks a clear loop back
Right now: browse → buy → done.  
Should be: buy → own → relist / use / import to Astra.
